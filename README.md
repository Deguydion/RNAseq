# RNAseq analysis of exposed butterfly brains. 
From Dion et al. Butterfly brains change in morphology and in gene splicing patterns after brief pheromone exposure. 

## Process
1. Adaptor trimming
2. Bacterial removal
3. Ribosomal removal
4. Read alignment
5. DEG analysis
6. DSG analysis
7. Annotation 
8. GO enrichment
